This is the paid service version of the free CLI.

The free tool can generate a first-pass score. A paid audit adds human review,
rewrite direction, and a client-ready HTML report that opens in any browser.

## What The Client Receives

- One browser-ready HTML report
- Score for each listing
- Title, description, and unit economics breakdown
- Risk phrases that may hurt trust, marketplace review, or ad approval
- Top priority actions for each listing
- Plain-language notes on where to improve title, bullets, margin, or ad spend

## Suggested Packages

### Starter Audit

Best for one product page or one marketplace listing.

- 1 listing reviewed
- HTML report
- Top 3 rewrite priorities
- Margin and break-even ad spend check

### Small Batch Audit

Best for a seller testing several products.

- 3 to 5 listings reviewed
- Bulk HTML report
- Priority list from weakest to strongest listing
- Listing-level risks and next actions

### Template Customization

Best for sellers who want to reuse the workflow.

- Custom CSV template
- Custom scoring notes for one niche or marketplace
- Example command and sample report

## What To Send

Only send non-sensitive information:

- Public listing link or copied listing text
- Product category
- Target market or country
- Sale price
- Landed product cost
- Seller-paid shipping or fulfillment cost
- Marketplace fee rate
- Typical ad spend per order
- Estimated refund rate

## What Not To Send

Do not send:

- Store passwords
- Payment credentials
- Ad account access
- Customer data
- Order exports
- Private supplier contracts

Approximate numbers are enough for a safe first audit.

## Important Note

This audit is decision support. It does not guarantee sales, ranking,
marketplace approval, ad approval, or profit.
